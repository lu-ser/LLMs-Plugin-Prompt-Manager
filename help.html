Troubleshooting

Buttons Not Appearing

If the save/load buttons don't appear on a supported site:

  • Refresh the page and wait a few seconds
  • Check if the site is enabled in Settings
  • Try clicking in the text input area to trigger detection
  • The extension may need time to detect dynamic page changes

Prompts Not Saving

  • Make sure you have text in the input field before clicking save
  • Check your browser's storage settings
  • Try reloading the extension in chrome://extensions/

Site Not Supported

  • Try adding the site manually in Settings
  • Use the context menu to save selected text as prompts
  • Contact support to request official support for the site

Tips & Best Practices

  • Organize with Tags: Use consistent tags to categorize your prompts
  • Descriptive Titles: Edit prompt titles to make them more searchable
  • Regular Backups: Export your prompts periodically for safekeeping
  • Use Search: Take advantage of the search function to find prompts quickly
  • Improve Iteratively: Use the improve feature to refine your prompts over time
